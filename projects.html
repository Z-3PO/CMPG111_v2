Progressive Projects

One project, seven stages. Each stage adds a new topic — by the end you've used everything. Type the code yourself (don't copy-paste!) to build muscle memory.

📝 How to Use This Page: Start at Stage 1 and work your way down. Each stage builds on the previous one. Try to code it yourself first using just the requirements, then check the solution. By Stage 7, you'll have a complete app using every concept in the course.
Stage 1 · Input/Output

💰 Budget Tracker v1 — Basic Calculator

Your task: Ask the user for their monthly income and a single expense description and amount. Display the remaining balance.

🔑 Concepts practiced: input(), float() conversion, arithmetic, f-string formatting with :.2f
✅ Stage 1 Code
income = float(input("Monthly income (R): "))
desc   = input("Expense description: ")
amount = float(input(f"Amount for {desc} (R): "))

balance = income - amount

print(f"\n--- Budget Summary ---")
print(f"Income:   R{income:.2f}")
print(f"{desc}: R{amount:.2f}")
print(f"Balance:  R{balance:.2f}")
Stage 2 · Decisions

💰 Budget Tracker v2 — Budget Warnings

Build on v1: After calculating the balance, give the user feedback. If they've spent more than 50% of their income, warn them. If the balance is negative, alert them they're over budget. Add a spending category: "Need" or "Want".

🔑 New concepts: if/elif/else, comparison operators, Boolean conditions
✅ Stage 2 Code
income = float(input("Monthly income (R): "))
desc   = input("Expense description: ")
amount = float(input(f"Amount for {desc} (R): "))
category = input("Category (need/want): ")

balance = income - amount
pct_spent = (amount / income) * 100

print(f"\n--- Budget Summary ---")
print(f"Income:   R{income:.2f}")
print(f"{desc} ({category}): R{amount:.2f} ({pct_spent:.0f}%)")
print(f"Balance:  R{balance:.2f}")

if balance < 0:
    print("⚠️  OVER BUDGET!")
elif pct_spent > 50:
    print("⚠️  High spending — over 50% of income")
else:
    print("✅ Looking good!")

if category == "want" and pct_spent > 30:
    print("💡 Tip: Your 'want' spending is over 30%.")
Stage 3 · Loops

💰 Budget Tracker v3 — Multiple Expenses

Build on v2: Instead of one expense, let the user enter multiple expenses until they type "done". Track the running total, count of expenses, and calculate the average expense.

🔑 New concepts: while loop, sentinel value, accumulator pattern, counter pattern
✅ Stage 3 Code
income = float(input("Monthly income (R): "))
total_spent = 0
count = 0

desc = input("Expense description (or 'done'): ")
while desc != "done":
    amount = float(input(f"Amount for {desc} (R): "))
    total_spent += amount
    count += 1
    print(f"  Running total: R{total_spent:.2f}")
    desc = input("Expense description (or 'done'): ")

balance = income - total_spent

print(f"\n--- Budget Summary ---")
print(f"Income:    R{income:.2f}")
print(f"Expenses:  {count} totalling R{total_spent:.2f}")
if count > 0:
    print(f"Average:   R{total_spent/count:.2f}")
print(f"Balance:   R{balance:.2f}")

if balance < 0:
    print("⚠️  OVER BUDGET!")
else:
    print("✅ Within budget")
Stage 4 · Functions

💰 Budget Tracker v4 — Organised with Functions

Refactor v3: Break the code into reusable functions:

  • get_income() → asks for and returns income
  • get_expenses() → collects expenses using a loop, returns the list of (desc, amount) tuples
  • calculate_summary(income, expenses) → returns (total_spent, balance, average)
  • display_report(income, expenses, summary) → prints everything
🔑 New concepts: def, parameters, return, scope, tuple return values
✅ Stage 4 Code
def get_income():
    return float(input("Monthly income (R): "))

def get_expenses():
    expenses = []
    desc = input("Expense (or 'done'): ")
    while desc != "done":
        amt = float(input(f"Amount for {desc}: R"))
        expenses.append((desc, amt))
        desc = input("Expense (or 'done'): ")
    return expenses

def calculate_summary(income, expenses):
    total = 0
    for desc, amt in expenses:
        total += amt
    balance = income - total
    avg = total / len(expenses) if expenses else 0
    return (total, balance, avg)

def display_report(income, expenses, summary):
    total, balance, avg = summary
    print(f"\n--- Budget Report ---")
    print(f"Income: R{income:.2f}")
    for desc, amt in expenses:
        print(f"  {desc}: R{amt:.2f}")
    print(f"Total spent: R{total:.2f}")
    print(f"Average: R{avg:.2f}")
    print(f"Balance: R{balance:.2f}")
    if balance < 0:
        print("⚠️  OVER BUDGET!")

# --- Main program ---
income = get_income()
expenses = get_expenses()
summary = calculate_summary(income, expenses)
display_report(income, expenses, summary)
Stage 5 · Files & Exceptions

💰 Budget Tracker v5 — Save & Load

Build on v4: Add the ability to:

  • Save expenses to a file budget.txt (one per line: desc,amount)
  • Load previous expenses from the file when the program starts
  • Handle the case where the file doesn't exist yet (first run)
  • Handle invalid number input with try/except
🔑 New concepts: file read/write, append mode, with statement, try/except, .strip(), .split()
✅ Stage 5 Code
def load_expenses(filename):
    expenses = []
    try:
        with open(filename, "r") as f:
            for line in f:
                parts = line.strip().split(",")
                expenses.append((parts[0], float(parts[1])))
        print(f"Loaded {len(expenses)} previous expenses.")
    except FileNotFoundError:
        print("No previous data. Starting fresh.")
    return expenses

def save_expenses(filename, expenses):
    with open(filename, "w") as f:
        for desc, amt in expenses:
            f.write(f"{desc},{amt}\n")
    print("Saved!")

def get_income():
    while True:
        try:
            return float(input("Monthly income (R): "))
        except ValueError:
            print("Please enter a number.")

def get_new_expenses():
    new = []
    desc = input("Expense (or 'done'): ")
    while desc != "done":
        try:
            amt = float(input(f"Amount for {desc}: R"))
            new.append((desc, amt))
        except ValueError:
            print("Invalid amount, try again.")
        desc = input("Expense (or 'done'): ")
    return new

# --- Main ---
expenses = load_expenses("budget.txt")
income = get_income()
expenses += get_new_expenses()

total = sum(amt for _, amt in expenses)
print(f"\nTotal: R{total:.2f} | Balance: R{income - total:.2f}")

save_expenses("budget.txt", expenses)
Stage 6 · Lists & Tuples

💰 Budget Tracker v6 — Smart Analytics

Build on v5: Add analysis features using list operations:

  • Display expenses sorted by amount (highest to lowest)
  • Find and highlight the biggest and smallest expense
  • Group expenses by category (store as list of tuples: (desc, amount, category))
  • Show the total spent per category
🔑 New concepts: list methods (sort, append), max()/min(), tuple unpacking in loops, list of tuples, in operator
✅ Stage 6 Code (analytics portion)
def show_analytics(expenses):
    if not expenses:
        print("No expenses to analyze.")
        return

    # Sort by amount (descending)
    sorted_exp = sorted(expenses, key=lambda e: e[1], reverse=True)

    print("\n--- Expenses (highest first) ---")
    for desc, amt, cat in sorted_exp:
        print(f"  {desc} [{cat}]: R{amt:.2f}")

    # Biggest and smallest
    amounts = [amt for _, amt, _ in expenses]
    biggest_idx = amounts.index(max(amounts))
    smallest_idx = amounts.index(min(amounts))
    print(f"\n💸 Biggest:  {expenses[biggest_idx][0]} — R{max(amounts):.2f}")
    print(f"🪙 Smallest: {expenses[smallest_idx][0]} — R{min(amounts):.2f}")

    # Total per category
    categories = []
    cat_totals = []
    for desc, amt, cat in expenses:
        if cat in categories:
            idx = categories.index(cat)
            cat_totals[idx] += amt
        else:
            categories.append(cat)
            cat_totals.append(amt)

    print("\n--- By Category ---")
    for i in range(len(categories)):
        print(f"  {categories[i]}: R{cat_totals[i]:.2f}")
Stage 7 · Database

💰 Budget Tracker v7 — Full Database Version

The final stage: Migrate from a text file to a SQLite database. Create a menu-driven program with all CRUD operations:

  • Table: expenses (id, description, amount, category, date)
  • Menu: (1) Add expense, (2) View all, (3) View by category, (4) Delete expense, (5) Monthly summary, (6) Quit
  • Include input validation and exception handling throughout
  • Use functions for each operation
🔑 New concepts: SQLite CRUD, parameterised queries, commit(), fetchall(), SQL WHERE, ORDER BY, SUM()
✅ Stage 7 Code
import sqlite3

def setup_db():
    conn = sqlite3.connect("budget.db")
    cur = conn.cursor()
    cur.execute("""CREATE TABLE IF NOT EXISTS expenses (
        id INTEGER PRIMARY KEY,
        description TEXT NOT NULL,
        amount REAL NOT NULL,
        category TEXT,
        date TEXT DEFAULT CURRENT_DATE
    )""")
    conn.commit()
    return conn, cur

def add_expense(cur, conn):
    desc = input("Description: ")
    try:
        amt = float(input("Amount (R): "))
        cat = input("Category (need/want): ")
        cur.execute("INSERT INTO expenses(description,amount,category) VALUES(?,?,?)",
                    (desc, amt, cat))
        conn.commit()
        print("Added!")
    except ValueError:
        print("Invalid amount.")

def view_all(cur):
    cur.execute("SELECT id,description,amount,category,date FROM expenses ORDER BY date DESC")
    rows = cur.fetchall()
    if rows:
        for eid, desc, amt, cat, date in rows:
            print(f"  [{eid}] {date} | {desc} ({cat}): R{amt:.2f}")
    else:
        print("No expenses yet.")

def view_by_category(cur):
    cat = input("Category: ")
    cur.execute("SELECT description,amount FROM expenses WHERE category=?", (cat,))
    rows = cur.fetchall()
    total = 0
    for desc, amt in rows:
        print(f"  {desc}: R{amt:.2f}")
        total += amt
    print(f"Total for '{cat}': R{total:.2f}")

def delete_expense(cur, conn):
    try:
        eid = int(input("ID to delete: "))
        cur.execute("DELETE FROM expenses WHERE id=?", (eid,))
        conn.commit()
        print(f"Deleted {cur.rowcount} record(s).")
    except ValueError:
        print("Invalid ID.")

def monthly_summary(cur):
    income = float(input("Monthly income (R): "))
    cur.execute("SELECT SUM(amount) FROM expenses")
    total = cur.fetchone()[0] or 0
    balance = income - total
    print(f"Income:  R{income:.2f}")
    print(f"Spent:   R{total:.2f}")
    print(f"Balance: R{balance:.2f}")
    if balance < 0:
        print("⚠️  OVER BUDGET!")

# --- Main ---
conn, cur = setup_db()

while True:
    print("\n1.Add  2.View All  3.By Category  4.Delete  5.Summary  6.Quit")
    ch = input("Choose: ")
    if   ch == "1": add_expense(cur, conn)
    elif ch == "2": view_all(cur)
    elif ch == "3": view_by_category(cur)
    elif ch == "4": delete_expense(cur, conn)
    elif ch == "5": monthly_summary(cur)
    elif ch == "6": break
    else: print("Invalid choice.")

conn.close()
print("Goodbye!")
