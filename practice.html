EasyInput/Output

Q1 · Personal Greeting

Ask the user for their name and age. Print: "Hello [name], you are [age] years old."

✅ Solution
name = input("Name: ")

HardInput/Output

Q6 · Payslip Calculator

Ask for an employee's name, hourly rate, and hours worked. Overtime (over 40 hours) is paid at 1.5× the normal rate. Display a formatted payslip showing regular pay, overtime pay (if any), and gross pay.

✅ Solution
name  = input("Employee name: ")
rate  = float(input("Hourly rate (R): "))
hours = float(input("Hours worked: "))

if hours > 40:
    regular_pay  = 40 * rate
    overtime_pay = (hours - 40) * rate * 1.5
else:
    regular_pay  = hours * rate
    overtime_pay = 0

gross = regular_pay + overtime_pay
print(f"\n--- Payslip for {name} ---")
print(f"Regular:  R{regular_pay:.2f}")
print(f"Overtime: R{overtime_pay:.2f}")
print(f"Gross:    R{gross:.2f}")
Only hours above 40 attract the 1.5× multiplier. The regular component is always capped at 40 hours.

Q12 · Leap Year Checker

Ask for a year. Determine if it's a leap year. A year is a leap year if it's divisible by 4, except centuries — which must be divisible by 400.

✅ Solution
year = int(input("Year: "))
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} is a leap year")
else:
    print(f"{year} is not a leap year")
Uses both and and or. Test: 2024 = leap, 1900 = not leap, 2000 = leap.
HardDecisions

Q13 · Electricity Tariff

An electricity company charges per kWh: first 100 kWh = R1.50, next 200 kWh (101–300) = R2.00, above 300 kWh = R3.00. Ask for usage and display the total cost with a R50 service fee.

✅ Solution
usage = float(input("kWh used: "))
if usage <= 100:
    cost = usage * 1.50
elif usage <= 300:
    cost = 100 * 1.50 + (usage - 100) * 2.00
else:
    cost = 100 * 1.50 + 200 * 2.00 + (usage - 300) * 3.00
total = cost + 50
print(f"Usage cost: R{cost:.2f}")
print(f"Service fee: R50.00")
print(f"Total: R{total:.2f}")
Tiered pricing: each tier is calculated separately. For 350 kWh: (100 × 1.50) + (200 × 2.00) + (50 × 3.00) + 50 = R750.

Q19 · Sentinel Average

Ask the user to enter numbers until they type -1. Calculate and display the average of all entered numbers (excluding the -1). Handle the case where no numbers were entered.

✅ Solution
total = 0
count = 0
num = float(input("Number (-1 to stop): "))
while num != -1:
    total += num
    count += 1
    num = float(input("Number (-1 to stop): "))
if count > 0:
    print(f"Average: {total/count:.2f}")
else:
    print("No numbers entered.")
Sentinel loop with priming read. Check count > 0 before dividing to avoid ZeroDivisionError.
MediumLoops

Q20 · Password Gatekeeper

The password is "python123". Give the user 3 attempts. If correct, print "Access granted". After 3 wrong attempts, print "Account locked".

✅ Solution
correct = "python123"
attempts = 0
logged_in = False
while attempts < 3:
    pw = input("Password: ")
    attempts += 1
    if pw == correct:
        print("Access granted")
        logged_in = True
        break
    else:
        print(f"Wrong. {3 - attempts} attempts left.")
if not logged_in:
    print("Account locked.")
break exits the loop immediately on success. The flag logged_in tracks whether we exited via break or exhausted all attempts.

HardFunctions

Q28 · Prime Checker

Write is_prime(n) that returns True if n is prime. Use it to print all primes from 2 to 50.

✅ Solution
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True

for num in range(2, 51):
    if is_prime(num):
        print(num, end=" ")
A prime has no divisors between 2 and n-1. The early return False exits as soon as we find one.

Q29 · Stats Tuple Return

Write analyze(numbers) that takes a list and returns a tuple of (average, highest, lowest). Use tuple unpacking to display each value.

✅ Solution
def analyze(numbers):
    avg = sum(numbers) / len(numbers)
    return (avg, max(numbers), min(numbers))

scores = [72, 85, 91, 68, 79]
average, highest, lowest = analyze(scores)
print(f"Avg: {average:.2f}, High: {highest}, Low: {lowest}")
Functions can return multiple values as a tuple. Tuple unpacking assigns each to a separate variable.

HardFiles/Exceptions

Q35 · High Score Finder

Read scores.txt (format: name,score per line). Find and display the player with the highest score. Handle missing file and invalid data.

✅ Solution
try:
    best_name  = ""
    best_score = -1
    with open("scores.txt", "r") as f:
        for line in f:
            try:
                n, s = line.strip().split(",")
                if int(s) > best_score:
                    best_score = int(s)
                    best_name  = n
            except ValueError:
                pass
    print(f"Leader: {best_name} ({best_score})")
except FileNotFoundError:
    print("scores.txt not found.")
Nested try/except: outer catches missing file, inner catches bad lines (e.g., missing comma). pass silently skips bad data.

Q36 · Word Count Function

Write word_count(filename) that returns the total word count of a text file. Handle missing file by returning 0.

✅ Solution
def word_count(filename):
    try:
        with open(filename, "r") as f:
            return len(f.read().split())
    except FileNotFoundError:
        print(f"'{filename}' not found.")
        return 0

print(f"Words: {word_count('essay.txt')}")
.split() with no argument splits on all whitespace and removes empties. Returning 0 in the error case lets the caller always get an int.

Q50 · CRUD Menu System

Build a menu: (1) Add student, (2) View all, (3) Update grade, (4) Delete student, (5) Quit. Use a students table (id, name, grade).

✅ Solution
import sqlite3
conn = sqlite3.connect("school.db")
cur = conn.cursor()
cur.execute("""CREATE TABLE IF NOT EXISTS students (
    id INTEGER PRIMARY KEY, name TEXT, grade REAL)""")
conn.commit()

while True:
    print("\n1.Add 2.View 3.Update 4.Delete 5.Quit")
    ch = input("Choose: ")
    if ch == "1":
        n = input("Name: ")
        g = float(input("Grade: "))
        cur.execute("INSERT INTO students(name,grade) VALUES(?,?)", (n,g))
        conn.commit()
    elif ch == "2":
        cur.execute("SELECT * FROM students")
        for r in cur.fetchall(): print(r)
    elif ch == "3":
        sid = int(input("ID: "))
        ng = float(input("New grade: "))
        cur.execute("UPDATE students SET grade=? WHERE id=?", (ng,sid))
        conn.commit()
    elif ch == "4":
        sid = int(input("ID: "))
        cur.execute("DELETE FROM students WHERE id=?", (sid,))
        conn.commit()
    elif ch == "5":
        break
conn.close()
HardDatabase

Q51 · Top 3 Scorers

Query a students table and display only the top 3 highest grades with rank labels (1st, 2nd, 3rd).

✅ Solution
cur.execute("""SELECT name, grade FROM students
    ORDER BY grade DESC LIMIT 3""")
ranks = ["1st", "2nd", "3rd"]
for i, (name, grade) in enumerate(cur.fetchall()):
    print(f"{ranks[i]}: {name} — {grade}")
ORDER BY ... DESC LIMIT 3 gets the top 3. enumerate() gives the index for rank labels.
